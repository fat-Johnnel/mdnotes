## 1.3 集合运算的性质 🧮

设$E$为全集，$A, B, C \in P(E)$。以下为集合运算的主要算律：

### 基本算律
- **幂等律**  
 $A \cup A = A$（1.3.1）  
 $A \cap A = A$（1.3.2）
- **结合律**  
 $(A \cup B) \cup C = A \cup (B \cup C)$（1.3.3）  
 $(A \cap B) \cap C = A \cap (B \cap C)$（1.3.4）
- **交换律**  
 $A \cup B = B \cup A$（1.3.5）  
 $A \cap B = B \cap A$（1.3.6）
- **分配律**  
 $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$（1.3.7）  
 $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$（1.3.8）

### 特殊集合的性质
- **同一律**  
 $A \cup \varnothing = A$（1.3.9）  
 $A \cap E = A$（1.3.10）
- **零律**  
 $A \cup E = E$（1.3.11）  
 $A \cap \varnothing = \varnothing$（1.3.12）
- **逻辑律**  
  **排中律**：$A \cup \sim A = E$（1.3.13）  
  **矛盾律**：$A \cap \sim A = \varnothing$（1.3.14）

### 其他重要算律
- **吸收律**  
 $A \cup (A \cap B) = A$（1.3.15）  
 $A \cap (A \cup B) = A$（1.3.16）
- **德摩根律**  
 $A - (B \cup C) = (A - B) \cap (A - C)$（1.3.17）  
 $A - (B \cap C) = (A - B) \cup (A - C)$（1.3.18）  
 $\sim(B \cup C) = \sim B \cap \sim C$（1.3.19）  
 $\sim(B \cap C) = \sim B \cup \sim C$（1.3.20）

### 补集与双重否定
- **补集性质**  
 $\sim \varnothing = E$（1.3.21）  
 $\sim E = \varnothing$（1.3.22）  
  **双重否定律**：$\sim \sim A = A$（1.3.23）

## 1.3 集合运算性质的重要结果 📚

### 包含关系与运算
1.$A \cap B \subseteq A$，$A \cap B \subseteq B$（1.3.24）
2.$A \subseteq A \cup B$，$B \subseteq A \cup B$（1.3.25）
3.$A \subseteq A$（1.3.26）
4.$A - B = A \cap \sim B$（1.3.27）

### 集合等价关系
5.$A \cup B = B \Leftrightarrow A \subseteq B \Leftrightarrow A \cap B = A \Leftrightarrow A - B = \varnothing$（1.3.28）

### 对称差运算
6. **交换律**：$A \oplus B = B \oplus A$（1.3.29）  
7. **结合律**：$(A \oplus B) \oplus C = A \oplus (B \oplus C)$（1.3.30）  
8. **空集性质**：$A \oplus \varnothing = A$（1.3.31）  
9. **自反性**：$A \oplus A = \varnothing$（1.3.32）  
10. **唯一性**：$A \oplus B = A \oplus C \Rightarrow B = C$（1.3.33）

## 1.4 有穷集的计数 📚

### 定理 1.4.1（包含排斥原理）
设$S$为有穷集，$P_1, P_2, \ldots, P_n$是$n$条性质。令$A_i$表示$S$中具有性质$P_i$的元素子集，则 **不具有所有性质** 的元素个数为：  
$$
|\overline{A_1} \cap \overline{A_2} \cap \cdots \cap \overline{A_n}| = |S| - \sum_{i=1}^n |A_i| + \sum_{1 \leq i < j \leq n} |A_i \cap A_j| - \cdots + (-1)^n |A_1 \cap A_2 \cap \cdots \cap A_n|
$$

### 推论 1.4.1
**至少具有一条性质** 的元素个数为：  
$$
|A_1 \cup A_2 \cup \cdots \cup A_n| = \sum_{i=1}^n |A_i| - \sum_{1 \leq i < j \leq n} |A_i \cap A_j| + \cdots + (-1)^{n-1} |A_1 \cap A_2 \cap \cdots \cap A_n|
$$

## 例 1.4.3 错位排列的计数问题 🤔
**问题**：$n$个人随机领取帽子，求每人都没领到自己帽子的方式数$D_n$。  
**解**：  
设排列$i_1 i_2 \ldots i_n$表示领帽方式，其中$i_j \neq j$（错位排列）。  
通过包含排斥原理可证：  
$$
D_n = n! \left( 1 - \frac{1}{1!} + \frac{1}{2!} - \cdots + (-1)^n \frac{1}{n!} \right)
$$
